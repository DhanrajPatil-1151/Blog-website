1. Temporary Block Method

You can use Google Search Console to temporarily block a website from being listed in Google search.

It is a very simple UI that accepts requests for blocking URL paths from your website. You have an option to block a single webpage or the whole website.

Webpage block or website block requests last only for 6 months. After that, your website will appear in Google search.

Blocking a URL does not prevent Google from crawling your page, only from showing it in search results.

2. Permanent Block Method

You can disallow crawlers from indexing certain parts of your website or the whole website by modifying your robots.txt file.

The robots.txt file is used to communicate with web crawlers. This file should be kept in the root directory. You can configure this file to prevent crawlers from indexing webpages on your site.

Use cases for blocking crawlers include:

  • Blocking indexing of certain parts of your website that have private information
  • Blocking indexing of a website in maintenance mode
  • Blocking indexing of web pages intended for internal or company use

To block all crawler bots from indexing all pages on your website, the robots.txt file will look like this:

User-agent: *
Disallow: /

In the User-agent field, you can specify specific bots like Googlebot, Bingbot, etc.

In the Disallow field, you can add specific routes from your website like /private/ or /private/blocked-page.html.
